## 🎯 О проекте

**Insurance CRM Pro** — это десктопное приложение для управления клиентской базой страховых компаний. Система позволяет вести учёт клиентов, полисов и предоставляет разграничение прав доступа между администраторами и обычными пользователями.

### Ключевые особенности:
- ✅ Современный интерфейс на базе Tkinter
- ✅ Безопасное хранение паролей (SHA256 хеширование)
- ✅ Ролевая модель доступа (Admin/User)
- ✅ Полный CRUD для клиентов
- ✅ Локальная SQLite база данных

## 📖 Использование

### Вход в систему

| Роль | Логин | Пароль |
|------|-------|--------|
| Администратор | `admin` | `admin123` |
| Пользователь | `user` | `user123` |

### Работа с клиентами

1. **Добавление**: Заполните поля в левой панели → нажмите **«Добавить»**
2. **Редактирование**: Выберите запись в таблице → измените поля → нажмите **«Изменить»**
3. **Удаление**: Выберите запись → нажмите **«Удалить»** → подтвердите действие

### Поля клиента

| Поле | Описание |
|------|----------|
| ФИО | Полное имя клиента |
| № Полис | Номер страхового полиса |
| Адрес | Адрес регистрации |
| Телефон | Контактный телефон |
| Email | Электронная почта |
| Уровень | Diamond/Platinum/Gold/Silver |
| Дата начала | Начало действия полиса (ДД.ММ.ГГГГ) |
| Дата окончания | Окончание действия полиса |

## 📁 Структура проекта

```
insurance-crm-pro/
├── main.py              # Точка входа, окно авторизации
├── auth.py              # Модуль аутентификации и БД
├── facade.py            # Основной интерфейс CRM
├── insurance.db         # SQLite база данных (создаётся автоматически)
├── README.md            # Документация
└── .gitignore           # Игнорируемые файлы
```

### Описание модулей

| Файл | Назначение |
|------|------------|
| `main.py` | Инициализация окна входа, запуск приложения |
| `auth.py` | Хэширование паролей, работа с таблицей пользователей |
| `facade.py` | GUI приложения, CRUD операции с клиентами |

## 🗄️ База данных

### Таблица `users`

| Поле | Тип | Описание |
|------|-----|----------|
| username | TEXT | Имя пользователя (PK) |
| password | TEXT | Хэш пароля (SHA256) |
| role | TEXT | Роль: 'a' (admin) / 'u' (user) |

### Таблица `clients`

| Поле | Тип | Описание |
|------|-----|----------|
| id | INTEGER | Уникальный идентификатор (PK) |
| name | TEXT | ФИО клиента |
| policy | TEXT | Номер полиса |
| address | TEXT | Адрес |
| phone | TEXT | Телефон |
| email | TEXT | Email |
| level | TEXT | Уровень обслуживания |
| start_date | TEXT | Дата начала |
| end_date | TEXT | Дата окончания |

## 🔧 Настройка

### Изменение учётных записей

Откройте `auth.py` и отредактируйте список пользователей:

```python
users = [
    ('admin', hash_password('admin123'), 'a'),
    ('user', hash_password('user123'), 'u'),
    # Добавьте новых пользователей здесь
]
```

### Изменение дизайна

Цветовая схема настраивается в `main.py` и `facade.py`:

```python
# Пример изменения цвета фона
login_win.configure(bg="#f0f2f5")  # Новый цвет
```
